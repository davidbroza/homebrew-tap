# davidbroza/homebrew-tap

[Homebrew](https://brew.sh) tap for [David Broza](https://davidbroza.dev)'s projects.

## davidcast

Keyboard-first launcher for macOS. Press `⌥ Space`, type, hit Enter.

```bash
brew install --cask davidbroza/tap/davidcast
```

Updates handled by the app's built-in updater — `brew upgrade` is a
no-op (`auto_updates true` in the cask).

[Source](https://github.com/davidbroza/davidcast) · [Latest release](https://github.com/davidbroza/davidcast/releases/latest)

## skillchart

Dashboard for Claude Code skills — shows what each one costs you in context tokens and flags duplicates / heavy descriptions.

```bash
brew install davidbroza/tap/skillchart
skillchart
```

[Source](https://github.com/davidbroza/skillchart)
